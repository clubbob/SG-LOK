# 생산계획 시스템 서비스 프로세스 (정의용)

[개요]
본 문서는 생산계획 시스템 도입을 위한 서비스 프로세스를 정의하기 위한 회의용 문서이다.
본 시스템은 다수의 판매 담당자, 단일 생산계획 관리자, 생산 관리자를 전제로 설계되었다.

--------------------------------------------------
## 1. 시스템 접속
- 사용자는 역할에 따라 시스템에 접속한다.
- 역할: 판매 담당자 / 생산계획 관리자 / 생산 관리자
- 생산계획 관리자의 기본 화면은 전체 생산 일정 캘린더이다.

--------------------------------------------------
## 2. 전체 생산 상황 확인 (생산계획 관리자)
- 모든 생산요청 및 생산계획을 캘린더로 확인
- 라인별 일정 표시
- 상태 구분: 정상 / 지연 / 완료
- 캘린더는 A4 한 페이지 기준 PDF로 출력 가능
- 생산 수량 및 공정 실적은 관리하지 않음

--------------------------------------------------
## 3. 생산요청 등록 (판매 담당자)
- 신규 생산요청 등록
- 입력 항목:
  · 제품명 (자유 입력, 임시 정보)
  · 수량
  · 생산요청일
  · 완료요청일
  · 생산이유 (주문-고객사명 / 재고)
- 품목코드 생성 없음
- 상태: 검토 대기

--------------------------------------------------
## 4. 생산요청 검토 (생산계획 관리자)
- 판매 담당자가 등록한 생산요청 검토
- 검토 항목:
  · 제품명 적정성
  · 완료요청일 현실성
  · 전체 일정 영향

--------------------------------------------------
## 5. 생산요청 확정 및 품목코드 생성 (생산계획 관리자)
- 생산요청 확정
- 제품명 기준 정식 품목 확정
- 이 시점에 품목코드 자동 생성
- 품목은 마스터로 관리

--------------------------------------------------
## 6. 생산요청 조정 및 재확정 (선택)
- 수량, 완료요청일, 생산이유 조정 가능
- 최초 요청은 이력으로 유지

--------------------------------------------------
## 7. 라인 등록 (생산계획 관리자)
- 확정된 생산요청에 라인 등록
- 라인 등록 시 시스템이 자동 생성:
  · 실제 생산개시예정일
  · 생산완료예정일
- 해당 시점부터 생산계획으로 전환

--------------------------------------------------
## 8. 생산계획 자동 생성
- 기준 정보:
  · 품목코드 / 품목명
  · 수량
  · 라인
  · 실제 생산개시예정일
  · 생산완료예정일

--------------------------------------------------
## 9. 생산현황 자동 판단 (시스템)
- 완료예정일 기준 상태 자동 판단
  · 정상 / 지연 / 완료

--------------------------------------------------
## 10. 생산계획 재조정
- 생산계획 관리자 또는 생산 관리자가 조정 가능
- 조정 항목:
  · 실제 생산개시예정일
  · 생산완료예정일
  · 라인
  · 우선순위
- 변경 이력 자동 저장

--------------------------------------------------
## 11. 생산 시작 처리
- 생산 관리자 또는 생산계획 관리자가 처리
- 실제 생산개시일 등록 (자동 입력 후 수정 가능)

--------------------------------------------------
## 12. 생산완료 처리
- 생산 관리자 또는 생산계획 관리자가 처리
- 실제 생산완료일 등록 (자동 입력 후 수정 가능)

--------------------------------------------------
## 13. 완료 계획 관리
- 완료 상태 유지
- 기본 수정 제한
- 필요 시 완료 취소 가능

--------------------------------------------------
## [핵심 정리]
- 판매 담당자: 생산요청
- 생산계획 관리자: 확정·계획·조정 책임
- 생산 관리자: 실제 시작/완료 기록
- 판단은 사람, 기록은 시스템
