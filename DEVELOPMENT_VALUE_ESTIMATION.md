# SG-LOK Work Flow 개발 가치 추정

## 실제 개발 시간 정보

- **개발 기간**: 1개월 (30일)
- **일일 투자 시간**: 4시간
- **총 개발 시간**: 30일 × 4시간 = **120시간**

## 프로젝트 분석 결과

### 1. 기술 스택
- Next.js 15.5.7 (App Router)
- React 19.1.2
- TypeScript 5
- Firebase (Firestore, Auth, Storage)
- Tailwind CSS 4
- jsPDF (PDF 생성)
- Zustand (상태 관리)

### 2. 코드 규모
- 총 코드량: 약 0.72MB (약 18,000-20,000 라인)
- 페이지 수: 29개
- 비동기 함수: 46개
- 컴포넌트: 다수

### 3. 주요 기능
- **생산관리**: 요청/승인, 일정 관리, Gantt 차트, 실시간 통계
- **성적서 관리**: Material Test Certificate 작성, PDF 생성(이미지 포함), 다중 제품 지원
- **사용자/관리자 대시보드**: 통계, 현황 모니터링
- **문의하기**: 파일 첨부, 이메일 알림
- **회원 관리**: 승인 시스템, 역할 기반 접근 제어

### 4. 기술적 복잡도
- **PDF 생성**: jsPDF, 한글 폰트 처리, 이미지 변환/임베딩
- **Gantt 차트**: 생산 일정 시각화
- **실시간 동기화**: Firebase 실시간 업데이트
- **파일 관리**: Firebase Storage 업로드/다운로드
- **이미지 처리**: Canvas 변환, base64 인코딩

## 개발 가치 추정

### 방법 1: 시간당 단가 기준

**중급 개발자 기준 (50,000-80,000원/시간)**
- 최소: 120시간 × 50,000원 = **6,000,000원**
- 최대: 120시간 × 80,000원 = **9,600,000원**
- 평균: 120시간 × 65,000원 = **7,800,000원**

**시니어 개발자 기준 (80,000-120,000원/시간)**
- 최소: 120시간 × 80,000원 = **9,600,000원**
- 최대: 120시간 × 120,000원 = **14,400,000원**
- 평균: 120시간 × 100,000원 = **12,000,000원**

### 방법 2: 프로젝트 복잡도 반영

이 프로젝트의 기술적 복잡도:
- PDF 생성 및 이미지 처리 (고난이도)
- Gantt 차트 구현 (중-고난이도)
- 실시간 데이터 동기화 (중난이도)
- Firebase 통합 (중난이도)
- 다중 역할 기반 접근 제어 (중난이도)

**복잡도 보정 계수: 1.2-1.5배**
- 중급 개발자: 7,800,000원 × 1.3 = **10,140,000원**
- 시니어 개발자: 12,000,000원 × 1.3 = **15,600,000원**

### 방법 3: 시장 가치 기준

동일 규모 프로젝트 시장 가격:
- 최소: **8,000,000원**
- 평균: **12,000,000원**
- 최대: **18,000,000원**

3. **시장 가치**
   - 동일 규모 프로젝트: 8,000,000원 ~ 18,000,000원
   - 기술 복잡도 고려: **10,000,000원 ~ 15,000,000원**
